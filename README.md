## üìà Dashboard Features & Visuals
- **Daily Booked Room Nights by Booking Platform**
  - Displays booking trends across travel portals (MakeMyTrip, Tripster, LogTrip, Journey, etc.)
  - Helps identify highest-performing online booking channels

- **Total Revenue by Booking Status**
  - Revenue breakdown for Checked Out, Cancelled, and No-Show bookings

- **Total Bookings Trend by Week, Month, Quarter, and Year**
  - Shows cyclic and seasonal changes in booking behavior

- **Revenue & Bookings by Room Class**
  - Identifies best-selling hotel room categories

- **Daily Booked Room Nights by Property Name**
  - Top hotels: Atiq Palace, Atiq Exotica, Atiq City, etc.

- **Total Bookings by Booking Platform**
  - Highest contribution: **55K bookings from Others**, followed by **MakeMyTrip 27K**

## üí° Key Insights
